# Stick Hero Game

A fun and addictive browser-based game where you help a stick hero cross platforms by stretching sticks to the right length.

## Game Overview

Stick Hero is an engaging arcade-style game built with HTML5 Canvas and vanilla JavaScript. Guide your hero across platforms by holding down to stretch a stick, then release to drop it and walk across. Hit the perfect zone for double points!

## Features

- **3 Lives System**: Get three chances before game over
- **Perfect Hit Bonus**: Land on the red perfect zone for double score
- **Sound Effects**: Dynamic audio using Web Audio API
- **Haptic Feedback**: Vibration support on mobile devices
- **Responsive Design**: Works on desktop and mobile devices
- **Animated Loading Screen**: Beautiful intro with progress bar
- **Background Music**: Procedurally generated background melody
- **Smooth Animations**: Fluid character movements and transitions

## How to Play

### Desktop
- **Hold down the mouse** to stretch out a stick
- **Release** to drop the stick
- Walk across to the next platform
- Land on the red perfect zone for double points

### Mobile
- **Tap and hold** to stretch out a stick
- **Release** to drop the stick
- Walk across to the next platform
- Land on the red perfect zone for double points

## Game Mechanics

- Start with 3 lives (hearts)
- Each successful crossing earns 1 point
- Landing on the perfect zone earns 2 points
- Missing a platform costs 1 life
- Game continues until all lives are lost
- Press **RESTART** button or **SPACE** key to play again
